Passo a Passo

1️⃣

Copie sua chave SSH

Use Ctrl+C para copiar a chave exibida acima.

2️⃣

Acesse as configurações de SSH do GitHub

Clique no link abaixo para ir direto para a página:

Ir para GitHub SSH Keys
3️⃣

Clique em "New SSH key"

Procure por um botão verde escrito "New SSH key" no canto superior direito da página.

4️⃣

Preenchimento do formulário

Campo Valor
Title Laptop-Development (ou identifique seu PC)
Key type Authentication Key
Key Cole a chave SSH pública (começando com ssh-ed25519)
5️⃣

Clique em "Add SSH key"

O GitHub vai pedir sua senha ou código de autenticação para confirmar.

✅ Chave Adicionada! Agora você pode fazer o push do seu projeto.

Fazer o Push do Projeto

Após adicionar a chave SSH, abra o PowerShell e execute:

cd "D:\Pastas de Programação treinamentos\ProjetoAPI\task_api" git push -u origin main
