# 🚀 API de Gerenciamento de Tarefas
## 📌 Descrição

API REST desenvolvida com FastAPI para gerenciamento de tarefas, permitindo criar, listar, atualizar e deletar tarefas.

## 🧠 Objetivo

Projeto criado com foco em aprendizado de desenvolvimento backend utilizando Python e boas práticas de API.

## 🛠️ Tecnologias utilizadas

* Python
* FastAPI
* SQLAlchemy
* SQLite

## ⚙️ Funcionalidades

* Criar tarefas
* Listar tarefas
* Atualizar tarefas
* Deletar tarefas

## ▶️ Como executar

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
## 📚 Aprendizados

* Estrutura de APIs com FastAPI
* Integração com banco de dados
* Padrão CRUD
* Organização de projeto backend
